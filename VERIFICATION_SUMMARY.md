## 🔧 執行的修復

### 1. RulesSection.tsx 語法錯誤修復

**問題位置**: `components/RulesSection.tsx` 第 267 行

**問題描述**: JSX 文本中的 `->` 箭頭符號在 React 中需要特殊處理

**修復前**:
```tsx
<p><strong>規則：</strong>轉劍玉決定先後。A抽招 -> A做(3次機會) -> A成功B失敗則A得1分。</p>
```

**修復後**:
```tsx
<p><strong>規則：</strong>轉劍玉決定先後。A抽招 {'->'} A做(3次機會) {'->'} A成功B失敗則A得1分。</p>
```

**驗證**: ✅ TypeScript 編譯無誤
